Install:

Download and run Assseq-Setup.exe

- The setup program will default installation to directory "C:\Program Files (x86)\Assseq", and also ask if you want to create file associations for
alignment type files.


----*** ALTERNATIVE ***----
In directory "without_installer_version" is the program provided either as an .exe-file or a jar-file:

Assseq.exe or assseq.jar (just download and run)
